## 🎯 重要提醒

### 身份认证关键点
1. **禁止硬编码用户信息** - 必须从后端API获取真实用户数据
2. **统一API URL格式** - 避免末尾斜杠不一致导致401错误
3. **JWT Token验证** - 确保sub字段与登录用户名一致

### 开发检查清单
- [ ] 检查是否存在硬编码用户信息
- [ ] 验证API URL格式一致性
- [ ] 确认JWT token生成正确
- [ ] 测试不同用户登录流程
- [ ] 验证权限验证逻辑
